  Settings
Specifies the resolution of the source image. The optimal size will be automatically selected when an image is loaded.

64   =   64 × 64 canvas
128   =   128 × 128 canvas
256   =   256 × 256 canvas
512   =   512 × 512 canvas
Source Image Resize
Controls whether smoothing is applied when resizing source images.

On: Produces a smoother appearance (antialiasing).
Off: Results in a sharper, more pixelated look.
Resize Smoothing
Specifies the colour mode used when generating the emblem.

Low: Uses a reduced colour palette (smaller server request size).
Full: Retains the full colour range from the source image (larger server request size).
Colour Mode
Specifies the method used for generating the emblem. Depending on the structure of the source image, selecting either rows or columns may result in a smaller server request size.

Auto: Automatically calculates both horizontal and vertical methods, selecting the most efficient option (slower).
Rows: Processes the image horizontally.
Columns: Processes the image vertically.

The Auto option optimizes the method dynamically but may take longer to process larger images on some systems.
SVG Creation Method
Specifies the number of decimal places used for defining elements in the emblem SVG.

Low: 3 decimal places (recommended).
Medium: 4 decimal places.
High: 5 decimal places.

Increase this setting only if you notice visible tearing in the emblem, such as gaps or misalignment between colors.
